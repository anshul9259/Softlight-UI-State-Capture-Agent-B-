# Softlight UI State Capture System (Agent B)

This project automates workflows in web apps (Linear, Notion), captures each UI state, and builds a dataset.

## How to Run
1. Open `Softlight_Agent_B.ipynb` in Jupyter.
2. Run cells sequentially.
3. Screenshots and DOM snapshots appear in `runs/<timestamp>/<app>/<task>/`.
